# Skill Library

## 是什么

技能库用于管理本地安装的辅助技能，让 PPT Agent 在生成过程中按需参考模板、规则和静态建议。

## 当前能力

- 搜索技能
- 导入本地 zip
- 导入本地文件夹
- 导入 URL / GitHub 占位来源
- 启用技能
- 禁用技能
- 删除技能

## 安全原则

- 外部技能默认 `enabled=false`
- 外部技能默认 `trusted=false`
- 外部技能默认不执行脚本
- 外部技能默认不读取用户文件
- 外部技能默认不联网
- 外部技能默认不能获得 API Key

## 风险等级

- `low`：低风险
- `medium`：中风险
- `high`：高风险
- `unknown`：未知风险

高风险或未知风险技能默认不建议启用。

## 本地目录

- 开发模式：`skills/`
- Windows EXE：`%LOCALAPPDATA%/PPT Agent/skills/`

## 适用范围

当前版本中，技能主要提供：

- 静态模板
- 写作建议
- 结构约束
- 受控 patch 建议

技能不会直接覆盖你的 PPT 主流程结果。

## 如何删除技能

- 打开“技能库”
- 找到对应技能
- 点击“删除”

## 如何处理不可信技能

- 保持禁用
- 不要开启脚本执行
- 检查风险提示和来源
